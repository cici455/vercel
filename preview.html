Current Issue

The project uses Next.js 16.1.1, which has compatibility issues with:

  • Node.js v24.11.0 (too new for Next.js 16)
  • Turbopack (unstable in this version)

Solution

To run this project successfully, you need to:

  • Option 1: Use stable Next.js version
    • Run: npm install next@15 react@18 react-dom@18
    • Update package.json scripts to remove Turbopack references
    • Run: npm run dev
  • Option 2: Use static generation
    • Set output: "export" in next.config.ts
    • Run: npm run build
    • Run: npx http-server out
  • Option 3: Use a different framework
    • Consider migrating to Vite + React for better compatibility with Node.js v24
    • Or use Create React App with older Node.js version
